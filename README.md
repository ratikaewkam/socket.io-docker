# socket.io-docker
Build a chat app with Socket.IO, Node.js, and Docker

## Install Express
```bash
npm install express@4
```

## Install Socket.IO
```bash
npm install socket.io
```
---

## Pull command (Docker)
```bash
docker pull ratikaewkam/socketio-docker-app:v1.0
```
